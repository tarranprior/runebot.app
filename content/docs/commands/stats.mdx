The `/stats` command allows users to fetch player stats from the official hiscores.

## Usage

```text
/stats [username] [account_type (optional)]
```

### Example

```
/stats username: Zezima account_type: normal
```

Use the buttons below the embed to switch between **different activities**, such as <kbd>Boss Kills</kbd>, <kbd>Bounty Hunter</kbd>, and <kbd>Clue Scrolls</kbd>. You can also use the <kbd>⟳</kbd> button to **refresh stat data**.

### Delete Accounts

You can remove an account by selecting it from the dropdown and clicking the <kbd>Delete</kbd> button.

The bot will automatically set the next account in the list as the new default. This means if you don't have any other accounts, you'll need to add a new account with `/setrsn` before using `/stats` again.
